# 🐾 Petly - Your Desktop Task Companion

Petly is a modern, transparent desktop pet built with Electron and React. It helps you stay productive while keeping you company on your screen.

## ✨ Features

- **Interactive Pets:** Choose between different characters: **Cat**, **Miku**, and **Ghost**.
- **Task Management:** A built-in checklist to track your daily goals.
- **Voice Lines:** Each pet has unique English voice lines when clicked.
- **Smart Interaction:** 
  - **Left-Click:** Trigger happy animations and voice lines.
  - **Right-Click Drag:** Move your pet anywhere on the screen.
  - **System Tray:** Change pet types and toggle sounds via the context menu.
- **Modern UI:** Clean, minimalist design inspired by Next.js patterns.
- **Transparent Window:** The pet floats seamlessly over your desktop applications.

## 🚀 Installation

### For Users
1. Download the latest `Petly Setup 1.0.0.exe` from the [Releases](https://github.com/marlon14421/Petly/releases) page.
2. Run the installer. It will automatically install and launch Petly.
3. Access settings and pet selection through the system tray icon (bottom right of your taskbar).

### For Developers
If you want to run or build the project yourself:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/marlon14421/Petly.git
   cd Petly
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run in development mode:**
   ```bash
   npm run dev
   ```
